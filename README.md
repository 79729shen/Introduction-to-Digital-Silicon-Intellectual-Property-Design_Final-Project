## Project Overview｜專案概述

### English

**Sample-based Quantum Diagonalization (SQD)** is a hybrid quantum–classical workflow for estimating eigenvalues and eigenvectors of quantum operators from samples generated by a quantum circuit.  
In this repository, SQD is applied primarily to **quantum chemistry** tasks, including:

- benchmarking **SQD vs. VQE** on the same molecular system,
- testing **molecular Hamiltonian energy estimation** on larger molecules,
- studying **configuration recovery** and **bit-flip probability design**,
- exploring **adaptive epsilon** strategies for noise-aware post-processing,
- validating behavior on **hardware-oriented or hardware-backed experiments**,
- extending the workflow toward **larger / drug-like molecular systems**.

This repository is **not a polished Python package**.  
It is best understood as a **research workbench + hackathon archive + reproducibility repository** containing reusable code, exploratory notebooks, benchmark artifacts, and presentation materials.

### 中文

**SQD（Sample-based Quantum Diagonalization）** 是一種 hybrid quantum–classical 的工作流程，透過量子電路產生的 samples，配合 classical post-processing 來估計量子算符的本徵值與本徵向量。  
在這個 repository 中，SQD 主要被用於 **量子化學** 相關任務，包括：

- 在相同分子系統上比較 **SQD 與 VQE**，
- 測試 **較大型分子 Hamiltonian 的能量估計**，
- 研究 **configuration recovery** 與 **bit-flip 機率設計**，
- 探索具抗雜訊能力的 **自動 epsilon（ε）調整方法**，
- 在 **硬體導向 / 硬體實測** 條件下驗證方法表現，
- 延伸到 **大型 / 類藥物分子** 的探索性應用。

這個 repository **不是** 已包裝完成的 Python 套件，  
更適合被理解成一個 **研究工作台 + hackathon 成果存檔 + 可重現實驗庫**。

## Reported Results｜結果摘要

| Experiment | Reported result (EN) | 結果摘要（中文） |
|---|---|---|

## Workflow Summary｜工作流程摘要

### English

The reusable workflow represented by `SQD_Alain.py` can be summarized as follows:

1. **Build molecule with PySCF**  
   Define atoms, basis, symmetry, frozen orbitals, and molecular integrals.

2. **Prepare a chemistry-inspired circuit**  
   Build a Hartree–Fock reference state and a UCJ/LUCJ-style ansatz using `ffsim`.

3. **Transpile to a target backend**  
   Use Qiskit transpilation / layout selection for fake or real IBM backends.

4. **Sample from hardware or hardware-like workflow**  
   Run on a backend or load pre-saved bit arrays / counts.

5. **Post-process with SQD**  
   Use configuration recovery and diagonalization of the sampled subspace.

6. **Analyze convergence and occupancies**  
   Plot energy convergence and average orbital occupancies.

### 中文

`SQD_Alain.py` 所代表的可重用工作流程大致如下：

1. **用 PySCF 建立分子系統**  
   定義原子、basis、symmetry、frozen orbitals 與 molecular integrals。

2. **建立化學啟發式量子電路**  
   以 Hartree–Fock 作為 reference state，搭配 `ffsim` 構造 UCJ/LUCJ 類 ansatz。

3. **針對目標 backend 進行 transpile**  
   透過 Qiskit 對 fake / real IBM backend 進行 layout 與電路最佳化。

4. **從硬體或硬體導向流程取得 samples**  
   可直接執行 backend，或讀取已儲存的 bit array / counts。

5. **以 SQD 進行 post-processing**  
   包含 configuration recovery 與 sampled subspace diagonalization。

6. **分析收斂與 occupancies**  
   繪製能量收斂曲線與平均軌域占據圖。

## Security Notes｜憑證與安全

### English

This repository includes local IBM Runtime setup examples, so please follow these rules carefully:

- `IBM_account_setup_local.ipynb` uses **blank placeholders** for `token` and `instance`.
- `sqd_h2o.py` also keeps `token="  "` and `instance="  "` as placeholders.
- **Never commit real IBM credentials** into the repository.
- `SQD_Alain.py` can read local `Token.txt` and `CRN.txt` files if they exist on your machine.
- Keep `Token.txt`, `CRN.txt`, `.env`, and `.qiskit/` **outside version control**.
- Use `save_account()` only on a **trusted local machine**.

Recommended `.gitignore` entries:

```gitignore
Token.txt
CRN.txt
.env
.qiskit/
__pycache__/
.ipynb_checkpoints/
```

### 中文

本 repo 含有本機 IBM Runtime 設定範例，因此請務必注意：

- `IBM_account_setup_local.ipynb` 內的 `token` 與 `instance` 為 **空白 placeholder**。
- `sqd_h2o.py` 也保留 `token="  "` 與 `instance="  "` 作為 placeholder。
- **不要將真實 IBM 憑證 commit 進 repository**。
- `SQD_Alain.py` 若在本機找到 `Token.txt` 與 `CRN.txt`，會讀取它們。
- `Token.txt`、`CRN.txt`、`.env`、`.qiskit/` 都應該 **排除在版控之外**。
- `save_account()` 僅適合在 **受信任的本機環境** 中使用。

建議 `.gitignore` 至少包含：

```gitignore
Token.txt
CRN.txt
.env
.qiskit/
__pycache__/
.ipynb_checkpoints/
```
